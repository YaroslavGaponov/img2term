Img2term
=====================
print image to terminal

![Demo]
(https://github.com/YaroslavGaponov/img2term/blob/master/term.jpg)

Terminal
---------
```output
img2term -i test.jpg -h 5 -w 5
````

Example
---------
```javascript
var img2term = require('img2term');
img2term.clear();
img2term.print('./test.jpg', 3, 3);
```
